source code 

Returns the branch flow limit constraint. The real power flows at the from end the lines are related to the bus voltage angles by Pf = Bf * Va + Pfinj.

_const_pf_constraints(self, gn, base_mva)

source code 

Returns a linear constraint enforcing constant power factor for dispatchable loads.

The power factor is derived from the original value of Pmin and either Qmin (for inductive loads) or Qmax (for capacitive loads). If both Qmin and Qmax are zero, this implies a unity power factor without the need for an additional constraint.

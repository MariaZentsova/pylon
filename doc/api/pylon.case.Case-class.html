makeBdc(self, buses=None, branches=None)

source code 

The bus real power injections are related to bus voltage angles by:

       P = Bbus * Va + Pbusinj

The real power flows at the from end the lines are related to the bus voltage angles by:

       Pf = Bf * Va + Pfinj

       | Pf |   | Bff  Bft |   | Vaf |   | Pfinj |
       |    | = |          | * |     | + |       |
       | Pt |   | Btf  Btt |   | Vat |   | Ptinj |

Based on makeBdc.m from MATPOWER by Ray Zimmerman, developed at PSERC Cornell. See http://www.pserc.cornell.edu/matpower/ for more information.

Returns: tuple
B matrices and phase shift injection vectors for DC power flow.

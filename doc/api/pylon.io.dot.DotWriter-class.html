write(self, file_or_filename, prog=None, format='xdot')

source code 

Writes the case data in Graphviz DOT language.

The format 'raw' is used to dump the Dot representation of the Case object, without further processing. The output can be processed by any of graphviz tools, defined in 'prog'.

create(self, dotdata, prog='dot', format='xdot')

source code 

Creates and returns a representation of the graph using the Graphviz layout program given by 'prog', according to the given format.

Writes the graph to a temporary dot file and processes it with the program given by 'prog' (which defaults to 'dot'), reading the output and returning it as a string if the operation is successful. On failure None is returned.

Based on PyDot by Ero Carrera.
